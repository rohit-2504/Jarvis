Overview

Jarvis is an advanced utility tool designed to enhance productivity and user experience by integrating multiple functionalities into a single application.

Features

Arithmetic Operations

Perform basic arithmetic calculations including addition, subtraction, multiplication, and division. Input two numbers and select the desired operation to receive immediate results.

Dynamic Background Changes

User Preferences

Remember user names for personalized interactions. Store user preferences for background images and search commands across sessions.

File Creation

Create and manage files directly from the application. Save user-generated content or data in various file formats.

Display the current battery level of the device within the app. Keep track of battery status to manage power efficiently.

User Interface

Intuitive and user-friendly design that integrates all features seamlessly. Responsive layout ensures a consistent experience across devices.
